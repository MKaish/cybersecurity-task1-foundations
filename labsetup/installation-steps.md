# Installation Steps

## Step 1: Install VirtualBox
Download and install VirtualBox/VMware.

## Step 2: Install Kali Linux
- Allocate 4GB RAM
- 2 CPU cores
- Host-Only Network

## Step 3: Install Metasploitable2
- Import VM file
- Use Host-Only Network

## Step 4: Verify Network
Run:
ping <target-ip>

Successful ping confirms connectivity.
